# Changelog — Il Forno Madre Website

## v1.0.0 — 2026-05-12

Prima versione della landing page istituzionale.

### Cosa include

- Landing page completa in HTML e CSS vanilla (file unico `index.html`)
- Design system con palette colori (paper, ink, terracotta, olive, copper)
- Tipografia Cormorant Garamond + Inter via Google Fonts
- Navigazione fixed con backdrop-filter blur
- Hero section con immagine e card sovrapposta
- Sezione brand con logo stage, note sul marchio, palette swatches
- Sezione filosofia con quote card sticky e manifesto testuale
- Sezione processo (tre card: Farine, Tempo, Mano)
- Sezione prodotti editoriale (pizza al taglio, basi pizza, focacce)
- Sezione differenziatori (quattro card)
- Experience band con immagine full-width
- Sezione B2B per professionisti
- Sezione contatti con dati aziendali completi
- Final CTA con radial gradient
- Footer con copyright e dati fiscali
- Responsive completo (breakpoint 980px e 560px)
- Hover animation su card prodotti (zoom 1.045x, 1s ease)
