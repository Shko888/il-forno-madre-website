# Il Forno Madre — Website

Landing page istituzionale per **Il Forno Madre**, laboratorio artigianale di pizza al taglio, basi pizza e focacce a Racconigi (CN).

## Stack

HTML e CSS vanilla. Nessun framework, nessuna dipendenza. Font via Google Fonts (Cormorant Garamond + Inter).

## URL

| Ambiente | URL |
|---|---|
| Produzione | [www.ilfornomadre.it](https://www.ilfornomadre.it) |
| Preview Vercel | [il-forno-madre.vercel.app](https://il-forno-madre.vercel.app) |

## Scopo

Questa è la landing page **istituzionale e di brand** del Forno Madre. Presenta identità, filosofia, prodotti e contatti.

È separata dalla piattaforma B2B (`il-forno-madre.vercel.app`), che gestisce ordini e relazioni con professionisti (ristoranti, gastronomie, bar evoluti).
